# KOMAD Furniture Website Project

## 🚀 Project Overview

This project is a responsive website developed for a furniture company, designed to showcase technical skills in front-end development using **HTML, CSS, and JavaScript**. The primary objectives were to:
- Demonstrate proficiency in creating **fully responsive layouts** across devices (desktop, tablet, mobile).
- Implement interactive components like modals, carousels, and dynamic navigation.
- Prioritize performance, accessibility, and clean code organization.
- Utilize modern CSS features (Flexbox, Grid) and vanilla JavaScript for functionality.

## ✨ Key Features

### Responsive Navigation Bar
- Dynamic menu adapting to screen sizes.
- Smooth scrolling for anchor links.
- Active state indicators for current page sections.

### Interactive Components
- **Image Carousel**: Autoplay and manual controls for showcasing products.
- **Modal Windows**: Lightbox-style pop-ups for detailed image viewing.
- **Testimonials Section**: User feedback display with great design.

### Layout & Styling
- Custom CSS Grid and Flexbox layouts for galleries and content sections.
- Consistent theming (colors, typography) using CSS variables.

### Additional Functionality
- Footer with embedded social media links and address map integration.

## 🛠 Technologies Used
- **HTML5** (semantic markup)
- **CSS3** (Flexbox, Grid, transitions, media queries)
- **JavaScript (ES6+)** (DOM manipulation, event handling)
- **SCSS** (Variables and nesting)
